# A Little Bookshelf

**[View the bookshelf →](https://diprekshya.github.io/LittleBookshelf)**

---

I built this project to learn how web scraping works in practice. I used Python to scrape the [Open Library](https://openlibrary.org) and pull real book data — titles, authors, genres, cover images, ratings, and publication years — for up to 1,000 books. That data then gets loaded into a clean, browsable library interface I built from scratch.

**What I learned:**
- How to scrape and clean real-world data from a public API
- How to structure and store data as JSON for use on the frontend
- How to build filtering, search, and sort functionality in vanilla JavaScript
- How localStorage works for persisting user data (reading lists) without a backend
- How to design a UI that makes a large dataset feel easy to explore
